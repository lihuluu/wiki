# 品牌重塑中的命名与品类扩展

## 核心问题

当品牌名与单一产品强绑定（如 Cocofloss = 牙线），品类扩展时会遇到天然天花板。品牌需要在**保留既有资产**和**打开未来空间**之间找到平衡。

## 从 Cocolab 案例提炼的框架

### 何时需要更名

| 信号 | 说明 |
|------|------|
| 品牌名 = 品类名 | 消费者无法联想到其他品类 |
| 产品线已超过核心品类 | 实际业务已超出名称承载范围 |
| 渠道升级受阻 | 名称限制了进入高端渠道的可能性 |
| 定价与品牌感不匹配 | 包装视觉与价格带产生认知冲突 |

### 更名后要做的五件事

1. **重新定义品类语言** — Cocolab 用 "Smile Care" 代替 "Oral Care"
2. **升级视觉系统** — 色彩、字体、图形风格全面刷新
3. **统一多品类设计语言** — 建立跨品类的系统性视觉规范
4. **锚定新的情绪定位** — 找到一个情绪词（如 "Dopamine"）贯穿全系统
5. **保留核心 DNA** — 不是推倒重来，而是"收紧 + 精致化"

### 命名的品类天花板

- **品类绑定型**：Cocofloss, Coca-Cola（扩展受限）
- **情感/价值观型**：Patagonia, Apple（扩展自由度高）
- **平台/系统型**：Cocolab, Lab Series（暗示体系感，扩展空间大）

## 包装设计的关键考量

品牌重塑时，包装是最直接触达消费者的载体：

- **色彩是质量信号**：柔和色调可能传达"可爱但廉价"，鲜明色彩传达"大胆与品质"
- **系统感 > 单品感**：多品类并排时必须像一个家族，不是拼凑
- **可持续性不能掉队**：品牌升级但包装材料降级，会产生信任危机
- **保留标志性元素**：Cocolab 保留了品牌符号和蓝色 DNA，确保老用户仍能识别
