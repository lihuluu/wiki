# 包装设计能力六层模型

> 包装设计的核心不是"美术能力"，而是：**在商业约束下，用视觉、结构、材料、信息层级和货架表现，影响消费者判断并降低购买决策成本。**

这个模型将包装设计师的能力从初级到高阶分为六个层次，帮助判断当前短板所在，并设定针对性提升目标。

---

## 1. 视觉基础层

**涵盖**：版式、字体、色彩、图形、插画、摄影、构图、信息层级

**常见症状**：
- 画面"满"但没有重点
- 字体选择靠感觉，缺少逻辑
- 颜色看起来脏、俗、廉价
- 产品名、卖点、品牌名之间层级混乱
- 包装单独看还行，放到货架上就弱

**训练目标**：不是"多看作品"，而是训练 **视觉判断力**

---

## 2. 品类理解层

**核心问题**：你在设计一个"漂亮盒子"，还是在设计一个**品类中的竞争武器**？

**不同品类的包装任务**：

| 品类 | 包装核心任务 |
|------|-------------|
| 零食 | 诱人、口味识别、冲动购买 |
| 护肤品 | 信任感、功效感、成分感、价格感 |
| 茶叶 | 产地、等级、礼品属性、文化感 |
| 宠物食品 | 营养可信度、宠物情绪连接、主粮/零食区分 |
| 酒类 | 品牌故事、质感、社交面子、价格锚定 |

**每做一个项目必须回答**：
- 这个品类的消费者在 3 秒内最先看什么？
- 这个品类的货架竞争规则是什么？
- 消费者凭什么相信它？
- 它应该显得便宜、亲民、专业、高端、天然、科技，还是稀缺？
- 它和竞品相比，第一眼差异在哪里？

---

## 3. 品牌策略层

**核心问题**：系列包装放在一起是否像一个品牌？

**需要提升的能力**：
- 品牌识别资产设计
- 系列化包装能力
- SKU 区分能力
- 主视觉延展能力
- 品牌调性一致性控制
- 不同价格带的视觉策略

**目标**：做出一个品牌可以长期使用、可以扩展、可以被消费者记住的**包装系统**

---

## 4. 结构与材料层

**核心问题**：你是否只会做效果图，不懂落地？

**真正的包装能力包括**：
- 盒型结构、开合体验
- 材料选择、印刷工艺、表面处理
- 成本控制、堆叠运输、陈列方式
- 环保与回收要求、实物落地误差控制

**成熟设计师必须知道**：
- 为什么这个盒型适合这个产品？
- 为什么用这类纸张/塑料/玻璃/铝材？
- 为什么这个工艺值得花钱？
- 哪些细节屏幕好看但实物会翻车？
- 哪些设计增加成本却不增加消费者感知价值？

**目标**：从"视觉包装设计师"升级为"**包装解决方案设计师**"

---

## 5. 商业转化层

**核心转变**：学会用商业指标评价设计，而不是只用审美评价

**关键指标**：
- 货架识别度、点击率、拿起率、复购率
- 客单价、价格感知、品牌记忆度
- 新老包装转化率差异
- 用户信任感、拍照分享意愿

**现实**：有些包装 Behance 上很漂亮，但超市里没有购买力。有些包装看起来"不够高级"，但卖得很好。

**目标**：让设计不仅能被同行夸，还能被消费者选择、被客户复购、被市场验证

---

## 6. 审美判断与风格掌控层

**高阶能力**：不是"会做某种风格"，而是能判断

- 什么风格适合这个品牌？
- 什么风格只是趋势，不值得跟？
- 什么元素会让产品显得廉价？
- 什么留白是真高级，什么留白只是空？
- 什么"国潮"是品牌资产，什么只是贴图案？
- 什么"极简"是克制，什么是没内容？

**训练方式**：大量案例拆解、实物观察、市场反馈和审美训练——**不是靠套模板**

---

## 与 5 类目标设定的对应关系

| 能力层 | 对应目标类型 |
|--------|-------------|
| 视觉基础 + 品类理解 | 作品目标、分析目标 |
| 结构与材料 | 技术目标 |
| 品牌策略 | 作品目标、输出目标 |
| 商业转化 | 商业目标 |
| 审美判断 | 输出目标 |
