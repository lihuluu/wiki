# Oishii

## 概述
Oishii 是全球最大的室内垂直草莓农场运营商，自 2018 年起销售无农药、非转基因的日本品种草莓。通过智能农场技术实现全年一致的甜度与品质。

## 包装创新：Nikko Berry Tray
为配合其最亲民的草莓品种 Nikko 的广泛分销，Oishii 推出了全新的顶部密封包装：

| 特性 | 详情 |
|------|------|
| 结构 | 7.5 oz 纸板托盘 + 轻质顶部薄膜密封 |
| 前身 | 传统塑料翻盖包装（clamshell） |
| 减塑 | 相较传统包装减少 80%+ 塑料用量 |
| 材料 | PET 薄膜直接密封于 SBS 纸板，无胶水或添加剂 |
| 功能 | 微穿孔控制氧气与水蒸气传输率，延长保鲜期 |

## 商业策略
- **新鲜度保证**：首次推出，邀请消费者在品质不达标时联系品牌
- **定价策略**：Nikko 是 Oishii 最实惠、分销最广的品种

## 可持续性争议
- **进步**：大幅减塑（按体积计算），在四十年未变的农产品包装类别中是实质性突破
- **局限**：PET 薄膜内衬在大多数美国市政不可路边回收；吸水垫需直接丢弃
- **展望**：团队正积极探索更可持续的替代方案

## 关键人物
