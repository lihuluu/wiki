### SK-01 需求解读与 Brief 拆解

```
┌────────────────────────────────────────────────┐
│  🃏 SKILL CARD  SK-01                          │
│                                                │
│  名称：需求解读与 Brief 拆解                     │
│  阶段：Phase 1 — 洞察与策略                     │
│  角色：设计主管 / 资深设计师                      │
│  前置技能：无（入口技能）                         │
│  关联技能：SK-02、SK-04                         │
└────────────────────────────────────────────────┘
```

#### 📌 技能定义

> 将客户的商业需求、模糊描述、口头信息，翻译为设计团队可执行的结构化 Brief 文档。

#### 📋 SOP 流程

```
Step 1 ── 接收原始需求
           ↓
Step 2 ── 使用「5W2H 提问框架」逐项确认
           ↓
Step 3 ── 识别隐性需求（客户没说但必须考虑的）
           ↓
Step 4 ── 输出《设计 Brief 确认书》
           ↓
Step 5 ── 客户签字确认 / 邮件回复确认
```

#### 🛠️ 工具与模板

**5W2H 提问清单：**

| 维度 | 必问问题 | 示例回答 |
|---|---|---|
| **What** 做什么 | 产品是什么？SKU 数量？ | 坚果礼盒，3 个 SKU |
| **Who** 给谁用 | 目标消费者？购买者 vs 使用者？ | 25-40 岁女性，送礼场景为主 |
| **Where** 在哪卖 | 销售渠道？线上/线下？ | 天猫旗舰店 + 山姆会员店 |
| **Why** 为什么做 | 改版原因？要解决什么问题？ | 老包装货架辨识度低，被竞品淹没 |
| **When** 时间节点 | 上市时间？倒推关键里程碑？ | 中秋节前 45 天交付量产文件 |
| **How** 怎么做 | 风格方向？有没有参考？ | 偏高端，参考某品牌调性 |
| **How much** 预算 | 单盒成本上限？设计费预算？ | 单盒成本 ≤ 12 元 |

**隐性需求清单（常被忽略的 12 项）：**

```
□ 是否需要适配电商主图拍摄角度
□ 是否有开箱体验要求（拆箱视频场景）
□ 物流跌落测试标准
□ 是否涉及出口（多语言/法规）
□ 是否需要系列化延展
□ 内包装与外包装是否分开供应商
□ 是否有已注册的品牌色/品牌字体
□ 上一版包装的消费者投诉反馈
□ 是否需要通过验厂/认证
□ 环保要求（FSC认证/可回收标识）
□ 是否考虑二次利用设计
□ 竞品定价区间（影响包装成本占比决策）
```

#### ⚠️ 常见失败模式

| 失败模式 | 后果 | 预防措施 |
|---|---|---|
| 只听业务转述，未直接对接客户决策人 | 后期反复推翻方案 | 要求 kick-off 会议有决策人参与 |
| Brief 未书面确认就开始设计 | 需求变更扯皮 | 必须有签字/邮件确认再开工 |
| 忽略渠道差异 | 电商和线下包装尺寸/信息重点不同 | 提问清单中明确区分渠道 |
| 没问清成本上限 | 设计了无法落地的方案 | 第一轮沟通就锁定成本区间 |

### SK-02 市场与竞品调研

```
┌────────────────────────────────────────────────┐
│  🃏 SKILL CARD  SK-02                          │
│                                                │
│  名称：市场与竞品调研                            │
│  阶段：Phase 1 — 洞察与策略                     │
│  角色：设计师 / 策略人员                          │
│  前置技能：SK-01                               │
│  关联技能：SK-03、SK-04                         │
└────────────────────────────────────────────────┘
```

#### 📌 技能定义

> 系统收集品类包装现状，分析竞品视觉策略、结构形式、货架表现，找到差异化机会。

#### 📋 SOP 流程

```
Step 1 ── 确定调研范围（品类、价格带、渠道）
           ↓
Step 2 ── 线上采集（电商平台截图、社交媒体开箱内容）
           ↓
Step 3 ── 线下采集（实地货架拍摄、购买实物样品）
           ↓
Step 4 ── 视觉拆解分析（色彩/版式/字体/材质/工艺）
           ↓
Step 5 ── 结构拆解分析（盒型/尺寸/开启方式）
           ↓
Step 6 ── 信息层级分析（正面信息优先级排列）
           ↓
Step 7 ── 绘制「包装竞争地图」
           ↓
Step 8 ── 输出《竞品调研报告》+ 机会洞察
```

#### 🛠️ 工具与方法

**竞品视觉拆解表：**

| 分析维度 | 竞品 A | 竞品 B | 竞品 C | 我方现状 |
|---|---|---|---|---|
| 主色调 | 红+金 | 深蓝+白 | 牛皮纸原色 | 橙色 |
| 版式风格 | 居中对称 | 左右分割 | 极简留白 | 满版铺图 |
| 主视觉元素 | 产品实拍 | 插画 | 无主图 | 产品实拍 |
| 字体调性 | 宋体-传统 | 黑体-现代 | 手写-文艺 | 黑体 |
| 信息焦点 | 品牌名 | 产品名+卖点 | 品牌故事 | 品牌名 |
| 材质 | 灰卡裱铜版 | 白卡 | 瓦楞原色 | 白卡 |
| 特殊工艺 | 烫金+击凸 | 局部UV | 无 | 覆哑膜 |
| 价格带 | ¥128 | ¥98 | ¥68 | ¥88 |

**包装竞争地图（二维定位）：**

```
        高端感
          ↑
          │      ◆ 竞品A
          │
 传统 ←───┼──────→ 现代
          │           ◆ 竞品B
          │   ★ 机会空间
          │
          ↓
        亲和力

（根据项目实际情况选择两个最关键的维度作为轴）
```

**货架模拟分析法：**

```
┌───┬───┬───┬───┬───┬───┬───┐
│ A │ A │ B │ B │ ? │ C │ C │  ← 实际/模拟货架排列
└───┴───┴───┴───┴───┴───┴───┘
                 ↑
            我方产品放入后
            是否能在 3 秒内被注意到？

测试方法：
1. 将所有竞品+我方设计排成货架
2. 邀请 5 人在 3 米外观看 3 秒
3. 问：你注意到了哪个？为什么？
```

#### 📦 案例：某坚果品牌竞品调研

```
【背景】
客户：某新锐坚果品牌，定价 ¥89-129，主攻线上+山姆
痛点：产品好但包装"长得像白牌"

【调研发现】
1. 行业色彩高度集中：80% 的坚果礼盒用红色+金色
2. 信息层级混乱：大部分竞品把"净含量"放得很大，品牌反而小
3. 结构雷同：90% 使用天地盖，开箱体验无差异
4. 消费者开箱视频分析：用户最常拍的画面 = 打开的瞬间

【机会洞察】
→ 颜色：避开红金，用深绿+米白打"健康高级感"差异
→ 信息：强化品牌名和产地故事
→ 结构：改用书型盒，打开有"翻开一本书"的仪式感
→ 针对开箱视频场景，设计内衬有惊喜元素

【最终成果】
上市 3 个月后天猫搜索「坚果礼盒」关键词，
该产品主图点击率比老包装提升 47%
```

| L1 了解 | L2 执行 | L3 熟练 | L4 专家 |
|---|---|---|---|
| 知道要做竞品调研 | 能按模板完成拆解表 | 能画出竞争地图并找到机会空间 | 能结合消费者数据做出有数据支撑的策略判断 |

#### 📌 技能定义

#### 📋 SOP 流程

```
Step 1 ── 明确目标人群画像
           ↓
Step 2 ── 收集消费者数据
           │  ├── 电商评论/问答区挖掘
           │  ├── 社交媒体关键词监测
           │  ├── 客服反馈/退货原因汇总
           │  └── 如有条件：焦点小组/问卷
           ↓
Step 3 ── 提炼包装相关的洞察
           ↓
Step 4 ── 输出《消费者洞察摘要》
```

#### 🛠️ 消费者画像模板

```
═══════════════════════════════════════
       消费者画像卡
═══════════════════════════════════════

👤 代表性人物：Emily，32 岁
💼 职业：互联网公司中层
💰 月收入：15-25K
🏠 城市：新一线
📱 常用平台：小红书、天猫

🎯 购买动机
   ├── 自用：注重成分、方便携带
   └── 送礼：要有面子、开箱惊喜

📦 对包装的期待
   ├── 不想要"土味"设计
   ├── 希望环保（加分项）
   ├── 拿到手要想拍照发朋友圈
   └── 文字要清晰，不想花时间找信息

😤 对包装的不满
   ├── "过度包装，里面东西少得可怜"
   ├── "打开后不知道怎么保存"
   └── "盒子扔了可惜，留着没用"

💡 设计机会
   → 社交货币感（值得拍照分享）
   → 功能性设计（可重复密封/二次利用）
   → 减少过度包装的负罪感
═══════════════════════════════════════
```

#### 🔍 电商评论挖掘技巧

```
关注三类评论：

1.「差评中提到包装的」
   → 真实痛点，直接对应改进点
   → 例："包装太大了以为量很多，打开发现就一小袋"

2.「好评中主动夸包装的」
   → 说明这个点真正打动了消费者
   → 例："盒子太好看了舍不得扔，拿来装首饰了"

3.「晒图/视频」
   → 分析消费者选择拍的角度 = 他们认为最好看的角度
   → 这个角度就是你电商主图的拍摄角度参考
```

### SK-04 包装策略定位

```
┌────────────────────────────────────────────────┐
│  🃏 SKILL CARD  SK-04                          │
│                                                │
│  名称：包装策略定位                              │
│  阶段：Phase 1 — 洞察与策略                     │
│  角色：设计主管 / 品牌策略                        │
│  前置技能：SK-01、SK-02、SK-03                  │
│  关联技能：SK-05、SK-12                         │
└────────────────────────────────────────────────┘
```

#### 📌 技能定义

> 基于 Brief、竞品调研和消费者洞察，确定包装设计的核心策略方向——在"对谁说、说什么、怎么说"上形成共识。

#### 📋 策略定位公式

```
包装策略 = 品牌定位 × 品类特性 × 渠道要求 × 竞争差异化

输出一句话定位：
"在（渠道）让（目标人群）看到后立刻感受到（核心价值），
 并通过（差异化手段）区别于（竞品/品类惯例）。"

示例：
"在山姆货架上让 30 岁新中产妈妈一眼识别出'高品质坚果'，
 通过深绿色系+书型盒结构 区别于市面千篇一律的红金色天地盖。"
```

#### 📋 策略方向决策矩阵

```
请在以下维度上做出选择（打✓）：

风格调性
□ 高端奢华   □ 品质简约   □ 年轻活力   □ 国潮文化   □ 自然有机

信息优先级（排序 1-5）
__ 品牌名/LOGO
__ 产品名称
__ 核心卖点/功能
__ 产地/原料故事
__ 促销/规格信息

视觉主角
□ 产品本身（实拍/插画）
□ 场景氛围
□ 品牌符号/图形
□ 纯文字排版

色彩策略
□ 跟随品类色（融入货架）
□ 对抗品类色（跳出货架）
□ 延续品牌色

结构策略
□ 成本优先（标准盒型）
□ 体验优先（特殊结构）
□ 功能优先（便携/可重复使用）
```

#### 🛠️ 5 种发想方法

**方法 1：关键词矩阵交叉法**

```
品牌关键词     ×    品类联想词     =   创意方向
─────────────────────────────────────────────
「探索」      ×    「坚果壳」     →   开壳的探索感/寻宝概念

「匠心」      ×    「烘焙过程」   →   烘焙工艺可视化

**方法 2：场景倒推法**

```
消费者使用场景 → 倒推包装应该是什么样

场景1：办公室下午茶分享
  → 包装要方便打开、可以放桌上、份量分装

场景2：过年送长辈

场景3：小红书晒图
  → 要有拍照构图的"画面感"、有文字金句可入镜

**方法 3：情绪板联想法**

**方法 4：反向思维法**

```

化妆品都方方正正     →  做圆柱/不规则

**方法 5：跨界嫁接法**

```
从其他行业"借"视觉语言：

奢侈品的包装语言 → 嫁接到茶叶包装

科技产品的极简感 → 嫁接到保健品包装

> 能准确绘制包装刀模图（Dieline），标注正确的切割线、压痕线、粘口位置，确保可直接用于开模生产。

#### 🛠️ 刀模图线型规范

```
标准线型约定（行业通用）：

────────── 实线（红色）     = 切割线 / 裁切
─ ─ ─ ─ ─ 虚线（绿色）     = 压痕线 / 折叠
─·─·─·─·─ 点划线（蓝色）   = 半切线
••••••••••• 点线（黄色）     = 粘口区域
─ ─ ─ ─ ─ 虚线（紫色）     = 出血线（通常外扩 3mm）

注意：
· 所有尺寸单位为 mm
· 标注内尺寸（产品实际占用空间）
· 纸张纹理方向需标注箭头
· 粘口宽度一般 15-20mm
```

#### 📋 刀模图绘制 SOP

```
Step 1 ── 确认产品尺寸（长×宽×高）+ 公差
           ↓
Step 2 ── 选择盒型 → 调出标准刀模模板
           ↓
Step 3 ── 根据产品尺寸调整模板参数
           │  ├── 内径 = 产品尺寸 + 间隙（通常1-2mm）
           │  ├── 纸厚补偿（350g白卡≈0.45mm）
           │  └── 压痕线位置校准
           ↓
Step 4 ── 标注所有尺寸、线型、纹理方向
           ↓
Step 5 ── 打印 1:1 白样，折叠验证
           ↓
Step 6 ── 修正后交付供应商确认
           ↓
Step 7 ── 供应商返回可生产确认 → 锁定刀模
```

#### ⚠️ 常见错误

| 错误 | 后果 | 预防 |
|---|---|---|
| 未考虑纸厚补偿 | 折叠后盒子变形 | 外层折线需补偿 1-2 个纸厚 |
| 压痕线位置偏移 | 折叠不整齐 | 1:1 打印验证 |
| 粘口太窄 | 粘合不牢 | 最少 15mm |
| 纹理方向未标注 | 折痕方向爆裂 | 折线尽量与纹理方向平行 |
| 未留出血 | 印刷后白边 | 四周各留 3mm 出血 |

```

#### 📌 技能定义

> 为包装选择合适的色彩方案，并确保从屏幕到印刷的色彩一致性。

#### 📚 包装色彩策略

```
策略一：品类色策略（融入货架）
─────────────────────────────────
牛奶 → 蓝白        咖啡 → 棕色系
茶叶 → 绿/红/金    巧克力 → 棕金
有机食品 → 绿色     儿童产品 → 多彩

优点：消费者快速识别品类
缺点：容易淹没在竞品中

策略二：对抗色策略（跳出货架）
─────────────────────────────────
当所有竞品都用红色 → 你用黑色
当品类习惯暖色调  → 你用冷色调

优点：货架辨识度极高
缺点：可能让消费者不认为你属于这个品类
建议：其他元素（产品图、文字）要强化品类归属

策略三：系列色策略（区分 SKU）
─────────────────────────────────
统一框架 + 每个口味/系列用不同色彩区分
要求：选色时考虑色彩之间的和谐性和区分度

选色工具：
· 色相环等距取色（3色→120°间隔，4色→90°间隔）
· 固定明度/饱和度，只变色相
· 考虑色盲人群的可区分性
```

#### 🛠️ 色彩管理流程

```
设计环节          注意事项
──────────────────────────────────────────────
屏幕设计时        · 使用校色过的显示器
                  · 在 AI/PS 中用 CMYK 模式工作
                  · 关键色指定 Pantone 色号

与客户确认时      · 提供 Pantone 色卡实物比对
                  · 告知"屏幕色≠印刷色"
                  · 关键色附 Pantone 编号

出文件时          · 所有色彩转为 CMYK
                  · 专色单独出分色版
                  · 黑色文字用单黑（K100）
                  · 大面积黑用四色黑（C60 M40 Y40 K100）

打样时            · 要求数码样+印刷样对比
                  · 关键色与 Pantone 色卡对比
                  · 在最终材质上打样（非铜版纸通用样）

量产时            · 提供签样样张随机跟色
                  · 约定 ΔE 容差范围（通常 ΔE<3）
                  · 首件确认后才批量生产
```

哑膜        哑光/高级     容易刮花，可加触感膜
触感膜      丝绒手感      成本较高，效果极佳
────────────────────────────────────────

二、烫印（Hot Stamping）
────────────────────────────────────────
类型        效果           注意事项
────────────────────────────────────────
烫金        金属光泽       线条不宜太细（≥0.3mm）
烫银        冷色金属感     同上
彩烫        多色金属       成本高于普通烫金
────────────────────────────────────────

三、压纹（Embossing / Debossing）
────────────────────────────────────────
类型        效果           注意事项
────────────────────────────────────────
击凸        凸起/立体感    纸张需有一定厚度
压凹        内凹/精致感    背面会有凸痕
─────────────────────────────────────────

四、UV 工艺
────────────────────────────────────────
类型        效果           注意事项
────────────────────────────────────────
局部UV      局部高光       与哑膜搭配效果最佳
磨砂UV      磨砂质感       触感好，成本中等
厚版UV      明显凸起触感   可做出类似"滴塑"效果
────────────────────────────────────────

五、其他
────────────────────────────────────────
工艺        效果           注意事项
────────────────────────────────────────
模切        造型/开窗      刀模精度影响效果
压花纹      纹理质感       需定制版辊
镭射        彩虹光         常用于防伪或视觉效果
植绒        绒面触感       成本高，用于局部
────────────────────────────────────────
```

#### 🛠️ 工艺组合搭配建议

```
高端经典组合：
哑膜 + 烫金 + 击凸 LOGO
→ 成本指数：★★★ | 高级感：★★★★★

性价比之选：
亮膜 + 局部 UV
→ 成本指数：★★ | 质感：★★★

年轻潮流组合：
触感膜 + 厚版 UV + 彩色烫印
→ 成本指数：★★★★ | 视觉冲击：★★★★★

极简高级组合：
无覆膜（裸感特种纸）+ 压凹 LOGO
→ 成本指数：★★★ | 独特性：★★★★★

注意事项：
· 烫金+UV 不能叠加在同一位置
· 覆膜后烫金附着力更好
· 击凸和烫金可以套位，但精度要求高
· 软包装的表面处理方式不同于纸盒
```

各环节占比参考（以中高端礼盒为例）：

#### 🛠️ 12 个降本不降质的方法

```
材料层面
1. 降低纸张克重（400g→350g 白卡，肉眼差异小）
2. 灰板厚度优化（2mm→1.5mm，非重物可用）
3. 用国产特种纸替代进口（省30-50%）
4. 统一材料规格，减少采购SKU

结构层面
5. 减少粘合工序（改用插口/卡扣代替糊盒）
6. 优化纸张开料（调整尺寸，减少废料率）
7. 一纸成型代替多部件组装
8. 取消不必要的内衬，改用结构卡位

工艺层面
9. 烫金面积缩小（LOGO烫金 vs 满版烫金差价巨大）
10. 用局部UV替代烫金（效果类似，成本低30%）
11. 减少工艺叠加（三道工艺→两道）
12. 合并印刷（几个SKU拼版同时印）
```

#### 🛠️ 卖稿技巧

```
10 个让客户买单的技巧：

1. 先讲"为什么这样做"，再展示"做了什么"
   → 策略先行，设计服务于策略

2. 用客户的语言，不用设计专业术语
   → "这个颜色让产品在货架上更醒目" ✅
   → "这是 Pantone 7740C 的低饱和度变体" ❌

3. 展示竞品对比图
   → 我们的方案放在竞品中间是什么效果

4. 模拟真实场景
   → 货架模拟图、电商页面模拟图、快递开箱模拟

5. 给客户选择题而非判断题
   → 提供 3 个方向，引导客户选择

6. 预设反对意见
   → "您可能会觉得颜色太大胆，让我解释为什么……"

7. 用数据说话
   → "竞品调研显示 80% 用红色，所以用绿色能跳出"

8. 展示细节诚意
   → 放大工艺细节、展示打开体验的动画/视频

9. 控制提案节奏
   → 不要一次全翻完，留悬念，逐步揭示

10. 结束时明确下一步
    → "如果确认方向 B，下周三我们可以交付深化稿"
```

## 如何使用这套 Skills 系统

| 场景 | 使用方式 |
|---|---|
| **新人入职** | 按 Phase 1→8 的顺序学习，每个 SK 阅读+实操 |
| **项目启动** | 按流程调出对应 SK 卡片的 Checklist 逐项执行 |
| **季度评估** | 用能力矩阵让员工自评+主管评估，画出雷达图 |
| **晋升考核** | 对照目标岗位的能力等级要求，找差距定学习计划 |
| **知识沉淀** | 每个项目完成后，将新经验补充进对应 SK 卡片 |
| **流程优化** | 定期回顾 SK 的 SOP 步骤，去掉冗余、补充遗漏 |
